# Belmond Landing Page

A premium, luxury landing page clone of Belmond.com, built with Next.js, Tailwind CSS, and Framer Motion.

## Features

- **Hero Video Section**: High-quality full-screen video background with crossing transitions.
- **Interactive Peru Map**: Custom interactive map showing Belmond destinations in Peru.
- **Premium Design**: Modern, responsive design with smooth transitions and elegant typography.
- **Language Switcher**: Integrated Google Translate functionality.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   cd belmond-landing
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment

### Deploy to Vercel

The easiest way to deploy your Next.js app is to use the [Vercel Platform](https://vercel.com/new).

1. Push your code to a GitHub repository.
2. Go to Vercel and import your repository.
3. Vercel will automatically detect Next.js and deploy your project.

### Environment Variables

If you have any environment variables, make sure to add them in the Vercel project settings.
